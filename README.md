# TalkellaBot - AI Social Media Content Bot

An AI-powered Telegram bot that generates ready-to-post social media content instantly.

## Features
- 🎁 3 FREE posts for new users
- 🤖 AI-powered content generation (Groq Llama 3.1)
- 💰 Pro: ₦2,000/month unlimited posts
- 📱 WhatsApp activation button
- 🇳🇬 Opay payment support

## Commands
- `/start` - Welcome message
- `/myid` - Get your Telegram ID
- `/status` - Check account status

## How to Run

### On Termux (Phone)
```bash
cd social-content-bot
pip install -r requirements.txt
python database.py
python bot.py
